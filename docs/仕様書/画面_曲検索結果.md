# 画面仕様書: 曲検索結果

## 1. 概要

| 項目 | 値 |
| --- | --- |
| パス | `/search/songs/:keyword` |
| 画面コンポーネント | `src/components/pages/search-songs.tsx` (`SearchSongsPage`) |
| ヘッダ | あり |
| URL パラメータ | `keyword` |
| 主目的 | 曲名でのキーワード検索結果を一覧表示 |

ヘッダ検索ボックスで検索タイプ「曲名」を選択し、検索ボタンを押すとこの画面に遷移する。URL は `/search/songs/${encodeURIComponent(keyword)}`。

## 2. データ取得

| 項目 | 値 |
| --- | --- |
| hook | `useSearchSongs(keyword)` |
| API | `GET /api/search/songs/:keyword` |
| レスポンス型 | `SongListResponse` |
| キャッシュ | なし |

```ts
interface SongListResponse {
    songs: SongListItem[];   // 各曲の id, name, artist, rank（rank は 0 固定）
    count: number;
}
```

`keyword` が変わるたびに再フェッチ。フロント側で `encodeURIWithPlus` をかけてから API を叩く（`+` 区切りの ufret.jp 形式に揃える）。

## 3. レイアウト

```
┌──────────────────────────────────────────────────────────────┐
│ Header                                                       │
├──────────────────────────────────────────────────────────────┤
│ ┃ 楽曲名 "{keyword}" の検索結果        全 N 件                 │
│                                                              │
│ ┌──────────────────────────────────────────────────┐         │
│ │  1. 曲名 - アーティスト                            │         │
│ │  2. 曲名 - アーティスト                            │         │
│ │  ...                                              │         │
│ └──────────────────────────────────────────────────┘         │
└──────────────────────────────────────────────────────────────┘
```

- `PageContainer` の中に `SectionTitle`（タイトル: `楽曲名 "{keyword}" の検索結果`、件数: `result?.count`）→ `ListContainer` → `NormalList`。
- 各行は `NormalListItem` + `SongListText`（曲名 + アーティスト名）。
- 行クリックで `/song/{song.id}` へ遷移。

## 4. ページネーション

**ページネーション無し**。`NormalList` でフラットに全件表示する。

> 補足: 同じ「検索結果」でもアーティスト検索結果画面はページネーション 15 件/ページがあるが、本画面にはない（実装上の選択）。

## 5. 状態と表示

| 状態 | 表示 |
| --- | --- |
| `loading === true` | 中央寄せの `CircularProgress` |
| `error` が truthy | 「取得に失敗しました」 |
| `result.songs.length` が 0 | 「取得に失敗しました」 |
| 通常 | 曲リスト |

`SectionTitle` の件数は `result?.count` を使う。

## 6. ナビゲーション

| 操作 | 遷移先 |
| --- | --- |
| 行クリック | `/song/{song.id}` |
| ヘッダ | [共通 UI・レイアウト仕様書 §3](./共通UI・レイアウト仕様書.md) |

`keyword` を変えて再検索する場合は、ヘッダの検索ボックスから別キーワードで検索する。
