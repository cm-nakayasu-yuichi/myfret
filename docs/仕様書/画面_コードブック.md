# 画面仕様書: コードブック

## 1. 概要

| 項目 | 値 |
| --- | --- |
| パス | `/codebook` |
| 画面コンポーネント | `src/components/pages/codebook.tsx` (`CodebookPage`) |
| ヘッダ | あり |
| URL パラメータ | なし |
| 主目的 | コードの押さえ方を一覧表示する画面（**現状はスケルトンのみ**） |

ヘッダメニューの「コードブック」項目からの遷移先。

## 2. 現在の実装

```tsx
export const CodebookPage = () => {
    return (
        <PageContainer>
            <SectionTitle>コードブック</SectionTitle>
        </PageContainer>
    );
};
```

- `PageContainer` の中に `SectionTitle`「コードブック」を 1 つ置くだけのスケルトン。
- API 連携・state・操作要素はいずれも未実装。

## 3. 状態と表示

`SectionTitle` のみが常に表示される（左に `palette.accent` の縦バー + 「コードブック」のテキスト）。
