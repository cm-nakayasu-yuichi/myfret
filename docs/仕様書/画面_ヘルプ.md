# 画面仕様書: ヘルプ

## 1. 概要

| 項目 | 値 |
| --- | --- |
| パス | `/help` |
| 画面コンポーネント | `src/components/pages/help.tsx` (`HelpPage`) |
| ルート定義 | `src/app/routes/help.tsx` |
| ヘッダ | あり |
| URL パラメータ | なし |
| 主目的 | アプリの使い方を表示する画面（**現状はスケルトンのみ**） |

ヘッダメニューの「ヘルプ」項目からの遷移先。

## 2. 現在の実装

```tsx
export const HelpPage = () => {
    return (
        <PageContainer>
            <SectionTitle>ヘルプ</SectionTitle>
        </PageContainer>
    );
};
```

- `PageContainer` の中に `SectionTitle`「ヘルプ」を 1 つ置くだけのスケルトン。
- API 連携・state・操作要素はいずれも未実装。

## 3. 状態と表示

`SectionTitle` のみが常に表示される（左に `palette.accent` の縦バー + 「ヘルプ」のテキスト）。
