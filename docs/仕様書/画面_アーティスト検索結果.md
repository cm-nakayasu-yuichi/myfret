# 画面仕様書: アーティスト検索結果

## 1. 概要

| 項目 | 値 |
| --- | --- |
| パス | `/search/artists/:keyword` / `/search/artists/:keyword/:page` |
| 画面コンポーネント | `src/components/pages/search-artists.tsx` (`SearchArtistsPage`) |
| ルート定義 | `src/app/routes/search-artists.tsx` |
| ヘッダ | あり |
| URL パラメータ | `keyword`、`page`（省略可） |
| 主目的 | アーティスト名でのキーワード検索結果をページネーション付きで表示 |

ヘッダ検索ボックスで検索タイプ「アーティスト名」を選択し、検索ボタンを押すとこの画面に遷移する。

## 2. データ取得

| 項目 | 値 |
| --- | --- |
| hook | `useSearchArtists(keyword)` |
| API | `GET /api/search/artists/:keyword` |
| レスポンス型 | `ArtistListResponse` |
| キャッシュ | なし |

```ts
interface ArtistListResponse {
    artists: ArtistListItem[];   // 各アーティストの id, name, rank（rank は 0 固定）
    count: number;
}
```

`keyword` が変わるたびに再フェッチ。フロント側で `encodeURIWithPlus` をかけて API を叩く。

## 3. レイアウト

```
┌──────────────────────────────────────────────────────────────┐
│ Header                                                       │
├──────────────────────────────────────────────────────────────┤
│ ┃ アーティスト名 "{keyword}" の検索結果      全 N 件             │
│                                                              │
│ ┌──────────────────────────────────────────────────┐         │
│ │ [pagination]                                     │         │
│ │  1. アーティスト名                                │         │
│ │  ...                                              │         │
│ │ 15. アーティスト名                                │         │
│ └──────────────────────────────────────────────────┘         │
└──────────────────────────────────────────────────────────────┘
```

- `PageContainer` の中に `SectionTitle`（タイトル: `アーティスト名 "{keyword}" の検索結果`、件数: `result?.artists.length`）→ `ListContainer` → `PaginatedList`。
- 各行は `NormalListItem` + `ArtistListText`。
- 行クリックで `/artist/{artist.id}` へ遷移。

> 補足: 件数表示は `result?.artists.length` を使う（曲検索結果が `result?.count` を使うのと異なる。値は同じはず）。

## 4. ページネーション

| 設定 | 値 |
| --- | --- |
| 1 ページの件数 | 15 |
| ベース URL | `/search/artists` |
| クエリ | `keyword`（URL に組み込む） |
| URL ページ番号 | `page` パラメータ |

```ts
usePagination(result?.artists || [], {
    itemsPerPage: 15,
    baseUrl: "/search/artists",
    currentUrlPage: page,
    query: keyword,
});
```

ページ URL は `buildPaginationUrl('/search/artists', keyword, page)` で組み立てる:

| ページ番号 | URL |
| --- | --- |
| 1 | `/search/artists/{keyword}` |
| 2 以降 | `/search/artists/{keyword}/{page}` |

### 4.1 キーボード操作

| キー | 挙動 |
| --- | --- |
| `ArrowLeft` | 前のページへ |
| `ArrowRight` | 次のページへ |

## 5. 状態と表示

| 状態 | 表示 |
| --- | --- |
| `loading === true` | `CircularProgress` |
| `error` が truthy | 「取得に失敗しました」 |
| `result.artists.length` が 0 | 「取得に失敗しました」 |
| 通常 | `PaginatedList` |

## 6. ナビゲーション

| 操作 | 遷移先 |
| --- | --- |
| 行クリック | `/artist/{artist.id}` |
| ページネーション | `/search/artists/{keyword}` または `/search/artists/{keyword}/{page}` |
| `ArrowLeft` / `ArrowRight` | 同上 |
